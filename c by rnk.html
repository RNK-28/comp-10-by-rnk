Act 2.4: Area of a Square

Question: Write a program that takes as input the length of one side of a square and calculates the area of the square.

Explanation: This program prompts the user for the side length of a square. It then calculates the area by multiplying the side length by itself and prints the result, formatted to two decimal places.

#include<stdio.h>

int main() {
    float side, area;
    printf("Enter length of one side of square: ");
    scanf("%f", &side);
    area = side * side;
    printf("The area of square is %.2f", area);
    return 0;
}

Act 2.5: Total Number of Balls

Question: Write a program that takes as input the number of balls in jar A and the number of balls in jar B. The program calculates and displays the total number of balls.

Explanation: The program asks for the number of balls in two separate jars. It adds these two numbers together and outputs the sum, representing the total number of balls.

#include<stdio.h>

int main() {
    int jarA, jarB, sum;
    printf("Enter the number of balls in jar A: ");
    scanf("%d", &jarA);
    printf("Enter the number of balls in jar B: ");
    scanf("%d", &jarB);
    sum = jarA + jarB;
    printf("Total Balls is: %d", sum);
    return 0;
}

Act 2.6: Shirt Discount Calculator

Question: Write a program that takes the original price of a shirt and a discount percentage from the user. The program should display the original price, the discount amount, and the final price after the discount.

Explanation: This program takes the original price and a discount percentage. It calculates the discount amount by multiplying the price by the percentage. It then subtracts this amount from the original price to get the final price, and displays all three values.

#include <stdio.h>

int main() {
    float original_price, discount_percentage, discount_amount, final_price;
    printf("Enter the original price of the shirt: ");
    scanf("%f", &original_price);
    printf("Enter the discount percentage: ");
    scanf("%f", &discount_percentage);

    discount_amount = original_price * (discount_percentage / 100.0);
    final_price = original_price - discount_amount;

    printf("\n--- Price Details ---\n");
    printf("Original Price: $%.2f\n", original_price);
    printf("Discount (%.0f%%): -$%.2f\n", discount_percentage, discount_amount);
    printf("Price after discount: $%.2f\n", final_price);

    return 0;
}

Act 2.7: Product of Digits

Question: Write a program that takes a 2-digit number from the user, computes the product of both digits, and shows the output.

Explanation: The program takes a two-digit number. It separates the first digit using integer division by 10 and the second digit using the modulo operator. It then multiplies these two digits and prints the result.

#include <stdio.h>

int main() {
    int number;
    int first_digit, second_digit;
    int product;
    printf("Enter a 2-digit number: ");
    scanf("%d", &number);

    if (number >= 10 && number <= 99) {
        first_digit = number / 10;
        second_digit = number % 10;
        product = first_digit * second_digit;
        
        printf("The number is %d\n", number);
        printf("The first digit is: %d\n", first_digit);
        printf("The second digit is: %d\n", second_digit);
        printf("The product of the digits is: %d\n", product);
    } else {
        printf("Invalid input. Please enter a valid 2-digit number.\n");
    }

    return 0;
}

Act 2.8: Seconds to Time Conversion

Question: Write a program that takes input in seconds and converts it into hours, minutes, and seconds.

Explanation: This program takes a total number of seconds. It uses integer division and the modulo operator to convert the seconds into hours, minutes, and the remaining seconds, then displays the converted time.

#include <stdio.h>

int main() {
    int totalSeconds;
    int hours, minutes, seconds;

    printf("Enter total seconds: ");
    scanf("%d", &totalSeconds);

    hours = totalSeconds / 3600;
    minutes = (totalSeconds % 3600) / 60;
    seconds = totalSeconds % 60;

    printf("Equivalent Time = %d hours, %d minutes, %d seconds.\n", hours, minutes, seconds);
    return 0;
}
